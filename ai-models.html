AI Laptop Price Predictor – Machine Learning Regression Project

A machine learning-powered laptop price prediction system built using Python, Pandas, and Scikit-learn. The model estimates laptop prices based on hardware specifications such as CPU, RAM, storage, GPU, screen size, and brand. Trained on over 2,000 real laptop listings and deployed as a live web app using Hugging Face Spaces and Gradio.

  • Built a complete machine learning workflow from data preprocessing to deployment
  • Trained on a real-world dataset containing 2,160 laptop listings
  • Handles data cleaning, missing values, and categorical feature encoding
  • Uses Random Forest Regression for price prediction
  • Achieved an R² score of approximately 0.83 on test data
  • Predicts prices from hardware specifications in real time
  • Supports CPU, RAM, storage, GPU, screen size, and brand inputs
  • Displays predicted prices in EUR, USD, and INR
  • Deployed publicly using Hugging Face Spaces
  • Interactive user interface built with Gradio

AI Spam Check – Machine Learning NLP Project

A lightweight AI-powered spam message classifier built using Python, Scikit-learn, and NLP techniques. This project detects whether a message is spam or not in real time and is deployed as a live web app using Hugging Face Spaces with a Gradio interface.

  • Built a complete machine learning pipeline from dataset to deployment
  • Uses Natural Language Processing (TF-IDF / Count Vectorization)
  • Trained using a Naive Bayes classification model
  • Real-time message prediction (Spam vs Not Spam)
  • Deployed as a live web application using Hugging Face Spaces
  • Interactive UI built with Gradio framework
  • Model trained on real SMS dataset for accurate classification
  • End-to-end project: training, testing, and deployment included
  • Optimized for fast inference and lightweight execution
  • First complete AI/ML deployment project in portfolio
